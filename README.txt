    Proyecto Cotizador de Automóviles

    Para este proyecto que etaré haciendo una simulación de un cotizador de autos, el usuario va a cotizar el modelo del fabricante de su auto por regiones que manejare 3 regiones que serán:

    - America
    - Europa
    - Asia

    Después tendra una opción para indicar de que año es el auto que el tiene. Después manejare la cotizacion por medio de un plan de seguro que tendré 2 (Básico y Completo)

    Es importante aclarar el porcentaje que tendrá cada auto de cada región el cual no estaré manejado con los siguientes datos: 

    - Americano = 1.15%
    - Europeo = 1.30%
    - Asiatico = 1.05%

    Entonces de acuerdo a la opción que el eliga en el año de su auto "le iremos restando un 3% por cada año". De igual manera dependiendo el plan de seguro que tenga se le aumentará un porcentaje: Si es básico se le aumentará el 30%, si es Completo se le aumentará el 50%.

    Tendrémos una interfaz de usuario donde si no esta validado algunos de los campos del formulario le aparecerá una alerta para que los llene. Una vez ya validado se le mostrará en la pantalla los datos de su auto con su respectiva cotización final.

    Para hacer este proyecto estaremos trabajando con Prototypes es necesario que ya tengan la noción de que es y para que se utilizan.

    Para el Maqueteado estere haciendolo con HTML5, Estilizado con CSS3 sin ningún framework y para la parte de programación usaré JavaScript.
